Tomcat的安装与启动

Posted on 2019-09-18 | In rjaz

Preface

JavaWeb实验课中有关Tomcat软件的安装与配置

下载

点击进入Tomcat官网
  如图点击下载：

安装

  解压安装包即可；建议文件地址中不要包含空格和中文。

启动

  进入bin目录，双击打开startup.bat

  等待一会，如图所示出现org.apache.catalina.startup.Catalina.start Server startup in 1184 ms即可；

  不要关闭该窗口，打开游览器输入localhost:8080,跳转到tomcat网页则说明安装成功。

Tomcat诊断

  在启动bin目录中的startup.bat脚本文件时，命令窗口一闪而过。在这中情况下无法查到错误信息，不能分析其出错的原因。

  这时可以先启动一个命令窗口，在这个命令行窗口中进入Tomcat安装目录中的bin目录，然后执行startup.bat命令，就会看到错误提示信息。

JAVA_HOME环境变量配置不正确

  如果出现下图的错误提示信息，则说明JAVA_HOME环境变量配置不正确，需要修改环境变量。

解决方案：  修改环境变量


端口8080被占用

  还有一种可能就是Tomcat服务器所使用的网路监听端口8080被其他服务程序占用了。怎么查看Tomcat的8080是否被占用了呢？

答案是通过查看日志文件，如图，出现了Address already in use: JVM_Bind就是表示端口号被占用了。

解决方案：  我们一般都是将Tomcat的默认端口号修改为80，这样不仅解决了端口占用的问题，还可以在访问Tomcat服务器时省略端口号不写。
