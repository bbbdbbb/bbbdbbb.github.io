标准输入输出和重定向

书上案例BinarySearch.class

package chapter1;

import java.util.Arrays;

import edu.princeton.cs.algs4.In;
import edu.princeton.cs.algs4.StdIn;
import edu.princeton.cs.algs4.StdOut;

public class X1_1_10_BinarySearch {

    public static void main(String[] args) {
        //将类型String的参数转化成Int类型
        int[] whitelist = In.readInts(args[0]);
        Arrays.sort(whitelist);
        while(!StdIn.isEmpty()) {
            int key = StdIn.readInt();
            if(rank(key,whitelist)<0)
                StdOut.println(key+"...");
        }
    }

    public static int rank(int key,int[] a) {
        //数组a[]必须是有序的
        int lo = 0;
        int hi = a.length-1;
        while(lo<=hi) {
            int mid = lo+(hi-lo)/2;
            if(key<a[mid])
                hi = mid-1;
            else if(key>a[mid])
                lo = mid + 1;
            else
                return mid;
        }
        return -1;
    }
}

代码解析

  这段代码中涉及到3个txt文件，每个文件有不同的作用，并且main函数与这3个txt文件交互的方式也不一样。这段程序用文件tinyW.txt作为白名单文件，将tinyW.txt的内容传给为main函数的参数args[];然后又将tinyT.txt中的内容作为标准输入传递给程序，程序会过滤掉tinyT.txt中所有存在于tinyW.txt中的元素，并将结果输出到控制台中；在这里，为了有更清楚的认识，我自己将程序的标准输入改为了文件temp.txt,重新执行程序，会把程序结果输出到temp.txt文件中。

将txt文件作为main函数的参数

  白名单文件tinyW.txt作为参数。设置main函数参数的方法如下：右击，选择Run As的Run Configurations,选择Arguments,在Program argument中输入txt文件的相对地址。[1]


  使用一组字符序列作为参数的方法如下，要修改参数的读取方式，用数组去读取参数args[]数组的内容。

将tinyT.txt文件重定向为标准输入

  文件tinyT.txt作为标准输入。设置标准输入的方法如下：右击，选择Run As的Run Configurations,选择Common,勾选input,中输入tinyT.txt文件的绝对地址[2]。

  执行程序，将在tinyT.txt文件中而不在tinyW.txt文字中的元素输出到控制台中。

将temp.txt文件重定向为标准输出

  在algs4-data文件中创建空的temp.txt文件，将其设置为标准输出。右击，选择Run As的Run Configurations,选择Common,勾选Output,中输入temp.txt文件的绝对地址。点击Run,执行程序。
